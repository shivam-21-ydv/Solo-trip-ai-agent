# SoloTrip AI — Multi-Agent Solo Travel Planning System

SoloTrip AI automates end-to-end solo travel planning using a multi-agent architecture built with CrewAI.
It collects user preferences, selects the best solo-friendly destination, estimates budget, creates day-by-day itinerary, and adds cultural & safety tips — without using ANY API keys.

## Features
✔ 5 sequential CrewAI Agents  
✔ Custom Mock Amadeus Travel Tool  
✔ Offline Gemini simulation (no API key required)  
✔ Long-term memory + Logging + Evaluation System  
✔ Runs 100% inside Kaggle safely
